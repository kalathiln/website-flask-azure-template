Java Streams Examples

This page provides examples of how to use Java Streams along with explanations for each operation.

1. Creating a Stream from a Collection

We can create a stream from any collection, such as a list or set, using the stream() method.

List<String> names = Arrays.asList("John", "Jane", "Jack", "Jill");
Stream<String> stream = names.stream(); // Create a stream from a list

2. Filtering a Stream

Using filter(), we can filter elements from a stream that match a condition, such as names that start with "J".

List<String> names = Arrays.asList("John", "Jane", "Jack", "Jill");
List<String> filteredNames = names.stream()
    .filter(name -> name.startsWith("J"))
    .collect(Collectors.toList());
// Output: [John, Jane, Jack, Jill]

3. Mapping a Stream (Transforming Data)

The map() function transforms each element of the stream, in this case, squaring numbers.

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
List<Integer> squares = numbers.stream()
    .map(n -> n * n) // Transform each number to its square
    .collect(Collectors.toList());
// Output: [1, 4, 9, 16, 25]

4. Reducing a Stream (Summing Values)

We can use the reduce() method to accumulate elements in the stream, such as summing numbers.

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
int sum = numbers.stream()
    .reduce(0, Integer::sum); // Summing all elements in the list
// Output: 15

5. Sorting a Stream

The sorted() method sorts the elements of a stream in natural order or a custom order.

List<String> names = Arrays.asList("John", "Jane", "Jack", "Jill");
List<String> sortedNames = names.stream()
    .sorted() // Sort the names in natural order
    .collect(Collectors.toList());
// Output: [Jack, Jane, Jill, John]

6. Distinct Elements from a Stream

The distinct() method removes duplicates from the stream.

List<Integer> numbers = Arrays.asList(1, 2, 2, 3, 4, 4, 5);
List<Integer> distinctNumbers = numbers.stream()
    .distinct() // Remove duplicates
    .collect(Collectors.toList());
// Output: [1, 2, 3, 4, 5]

7. Stream from an Array

We can also create a stream from an array using the Arrays.stream() method.

String[] fruits = {"Apple", "Banana", "Orange"};
Stream<String> fruitStream = Arrays.stream(fruits); // Create a stream from an array

8. Parallel Stream

Using parallelStream(), the stream operations are performed in parallel, leveraging multiple threads for performance.

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
numbers.parallelStream()
    .forEach(n -> System.out.println(n * n)); // Perform operations in parallel

9. Limit and Skip Operations

The skip() method skips a specified number of elements, and the limit() method limits the number of elements processed.

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6, 7, 8, 9);
List<Integer> limitedNumbers = numbers.stream()
    .skip(2) // Skip the first 2 elements
    .limit(3) // Limit to 3 elements
    .collect(Collectors.toList());
// Output: [3, 4, 5]

10. FlatMap (For Nested Collections)

flatMap() flattens nested collections, allowing us to process each element individually.

List<List<String>> listOfLists = Arrays.asList(
    Arrays.asList("a", "b", "c"),
    Arrays.asList("d", "e", "f")
);
List<String> flattenedList = listOfLists.stream()
    .flatMap(List::stream) // Flatten the nested lists
    .collect(Collectors.toList());
// Output: [a, b, c, d, e, f]
